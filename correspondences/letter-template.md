# Physical Mailer — Website Outreach

One-page letter. Print on standard 8.5×11, fold into a #10 envelope.
Address the envelope to "Owner / Manager" if no name is available.

The [DEMO URL] won't be clickable on print, so keep it short enough to
type. All demo URLs follow the pattern:
  florida-man-bioscience.github.io/demo-websites/generated-sites/[SLUG].html

Consider printing a QR code linking to [DEMO URL] in the top-right corner
— it dramatically increases the chance they actually pull it up.
