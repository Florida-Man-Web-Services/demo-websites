# Gainesville, FL — Businesses Without a Website on Google Maps

252 unique businesses across 42 categories. Each listing was confirmed to show no "Website" button on its Google Maps search result on June 16–17, 2026.

## Files

- `gainesville_no_website.xlsx` — Spreadsheet with filterable columns (name, search category, Google category, address, phone, rating, Maps link).
- `gainesville_no_website.json` — Same data as JSON.
- `google_maps_searches.txt` — The 50 Google Maps search URLs used (42 were successfully collected).

## Columns

| Field | Description |
| --- | --- |
| `name` | Business name as listed on Google Maps |
| `search_category` | The category we searched (e.g., `plumbers`, `bakeries`) |
| `category_label` | The category label Google Maps assigned to the business |
| `address` | Street address |
| `phone` | Phone number, if listed |
| `rating` | Star rating + review count |
| `google_maps_url` | Direct link to the business listing on Maps |

## Categories covered

restaurants, cafes, bakeries, food trucks, bars, hair salons, barbershops, nail salons, spas, auto repair, car wash, tire shops, plumbers, electricians, landscapers, lawn care, cleaning services, handyman, painters, roofers, HVAC, contractors, movers, pest control, daycares, dance studios, martial arts, gyms, jewelers, upholstery, print shops, real estate agents, insurance agents, accountants, notary, dry cleaners, laundromats, clothing boutiques, convenience stores, grocery stores, veterinarians.

## Not covered

locksmiths, thrift stores, pet groomers, tutors, photographers, event venues, florists, tailors, tattoo shops.

## Notes

- A small number of rows may have a website on the detail page even if the list view didn't show a Website button — worth a quick double-check before outreach.
- Some businesses appear in more than one related category (e.g., a contractor that also does painting). Exact duplicates were removed.
